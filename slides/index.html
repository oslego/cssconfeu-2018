What web designers need so they can
really design in the browser?

  • Freedom
  • Confidence
  • Specialized tools
  • Ergonomics
- Freedom: canvas that encourages free-from exploration by direct manipulation - Confidence: undo/redo, save changes, versioning - Specialized tools: code is an abstraction; slow to get from idea to visual means less experimentation - Ergonomics: customize, personalize, save and share pattern libraries
